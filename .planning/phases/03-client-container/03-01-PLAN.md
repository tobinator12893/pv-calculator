<task type="auto">
  <name>Task 1: Write nginx.conf and client Dockerfile</name>
  <files>pv-calculator/client/nginx.conf, pv-calculator/client/Dockerfile</files>
  <action>
Create `pv-calculator/client/nginx.conf` with ALL of the following (use the complete example from 03-RESEARCH.md):

1. `pid /var/run/nginx/nginx.pid;` — writable path for non-root nginx user
2. `worker_processes auto;` and `events { worker_connections 1024; }`
3. `http` block with `include /etc/nginx/mime.types;` (CRITICAL — without it JS files served as octet-stream)
4. Gzip block (NGINX-03): `gzip on; gzip_vary on; gzip_proxied any; gzip_comp_level 6; gzip_min_length 256;` with types: text/plain, text/css, text/xml, application/javascript, application/x-javascript, application/json, application/xml, application/rss+xml, image/svg+xml
5. `server` block listening on port 8080 (non-root cannot bind 80), `server_tokens off;`
6. Security headers (NGINX-05): X-Frame-Options SAMEORIGIN, X-Content-Type-Options nosniff, X-XSS-Protection, Referrer-Policy — all with `always` flag
7. Static asset cache (NGINX-04): `/assets/` regex location with `expires 1y; add_header Cache-Control "public, immutable";` and `/cesium/` prefix location with same cache headers
8. `index.html` exact match location with `no-cache, no-store, must-revalidate`
9. API reverse proxy (NGINX-02): `location /api/ { proxy_pass http://server:3001; }` — CRITICAL: NO trailing slash on proxy_pass (trailing slash strips /api/ prefix, breaking all Express routes). Include proxy_http_version 1.1, Host, X-Real-IP, X-Forwarded-For, X-Forwarded-Proto headers, proxy_read_timeout 90
10. SPA fallback (NGINX-01): `location / { try_files $uri $uri/ /index.html; }` with `add_header Cache-Control "no-cache";`
11. Hidden file deny: `location ~ /\. { deny all; }`

Do NOT include a `user` directive in nginx.conf — it is irrelevant when running as non-root via Dockerfile USER and produces a warning.

Then create `pv-calculator/client/Dockerfile` with two stages:

Stage 1 (builder):
- `FROM node:24-bookworm-slim AS builder`
- `WORKDIR /app`
- `COPY package.json package-lock.json ./` then `RUN npm ci` (BUILD-04: layer cache optimization)
- `COPY . .` then `RUN npm run build` (runs tsc -b && vite build)

Stage 2 (runtime):
- `FROM nginx:stable-alpine AS runtime`
- `RUN mkdir -p /var/run/nginx && chown -R nginx:nginx /var/cache/nginx /var/run/nginx /var/log/nginx /etc/nginx/conf.d` — MUST run before USER switch (BUILD-05)
- `COPY nginx.conf /etc/nginx/nginx.conf`
- `COPY --from=builder --chown=nginx:nginx /app/dist /usr/share/nginx/html` — chown is critical per Phase 2 lesson
- `USER nginx` (uid=101 built into nginx:stable-alpine)
- `EXPOSE 8080`
- `CMD ["nginx", "-g", "daemon off;"]`

Add `# syntax=docker/dockerfile:1` as first line.

<task type="auto">
  <name>Task 2: Build client Docker image and verify all requirements</name>
  <files>pv-calculator/client/Dockerfile, pv-calculator/client/nginx.conf</files>
  <action>
Build and verify the client Docker image. Run these commands in sequence:

1. Build: `docker build -t pv-client pv-calculator/client/`
   - Must complete without error (BUILD-02)

2. Run container: `docker run -d -p 8080:8080 --name pv-client-test pv-client`
   - Wait 2 seconds for nginx startup

3. Verify SPA fallback (NGINX-01):
   - `curl -o /dev/null -s -w "%{http_code}" http://localhost:8080/` — expect 200
   - `curl -o /dev/null -s -w "%{http_code}" http://localhost:8080/some-sub-path` — expect 200 (SPA fallback serves index.html)

4. Verify gzip (NGINX-03):
   - `curl -s -H "Accept-Encoding: gzip" -I http://localhost:8080/` — expect Content-Encoding: gzip
   - Find a JS file: `JS_FILE=$(docker exec pv-client-test ls /usr/share/nginx/html/assets/ | grep '\.js$' | head -1)`
   - `curl -s -H "Accept-Encoding: gzip" -I "http://localhost:8080/assets/$JS_FILE"` — expect Content-Encoding: gzip

5. Verify security headers (NGINX-05):
   - `curl -s -I http://localhost:8080/` — must contain X-Frame-Options: SAMEORIGIN and X-Content-Type-Options: nosniff

6. Verify cache headers (NGINX-04):
   - JS asset response must include `Cache-Control: public, immutable` and `expires` ~1 year
   - `curl -s -I http://localhost:8080/` should include `Cache-Control: no-cache` (SPA fallback location)

7. Verify non-root (BUILD-05):
   - `docker exec pv-client-test id` — expect uid=101(nginx)

8. Verify Cesium assets present:
   - `docker exec pv-client-test ls /usr/share/nginx/html/cesium/` — must show Workers, Assets, Widgets, ThirdParty

9. Verify nginx.conf syntax is valid:
   - `docker exec pv-client-test nginx -t` — expect "syntax is ok" and "test is successful"

10. Cleanup: `docker stop pv-client-test && docker rm pv-client-test`

Note on /api proxy (NGINX-02): Cannot test the reverse proxy standalone because Docker Compose DNS (`server:3001`) is not available without Compose networking. The proxy_pass config will be verified in Phase 5. For now, confirm the nginx.conf contains the correct `proxy_pass http://server:3001;` (no trailing slash) via grep.

If port 8080 is already in use on the host, use an alternative host port (e.g., -p 8081:8080) and adjust curl URLs accordingly.
